Opensource cursor engine

what's eggy?

'eggy' is opensource cursor engine,that generate single theme package. Designing,Compiling and Testing new cursor is slower process. Eggy wrap up tool and dependencies into standardized application that includes everything it need to build:package,design,sharing.

eggy is backed by

adwaita by gnome

bibata by fu1ie5_

dmz-black by GalliumOS

In Linux, every cursor ship with icon theme 😧
